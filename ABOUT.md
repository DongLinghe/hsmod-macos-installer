# About

HsMod macOS Reinject is an unofficial helper for restoring a working
BepInEx/HsMod injection setup after Hearthstone or Battle.net updates replace
the macOS app executable.

It is intentionally kept separate from HsMod itself because it does not change
HsMod gameplay features. It only manages the macOS launch wrapper and resource
layout needed for this tested setup.

Repository description suggestion:

```text
Unofficial macOS reinject helper for HsMod/BepInEx on Hearthstone.
```
